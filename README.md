## PHP Form Builder

A dynamic PHP Form Builder application that allows administrators to create custom forms, manage fields, collect submissions, and view submitted data through an admin dashboard.

## Features
1. Authentication
2. Admin Login System
3. Session-based Authentication
4. Protected Admin Routes
5. Logout Functionality

## Form Management
1. Create Forms
2. Edit Forms
3. Delete Forms
4. View All Forms

## Dynamic Field Builder
1. Text Field
2. Email Field
3. Number Field
4. Textarea
5. Dropdown
6. Radio Buttons
7. Checkboxes
8. File Upload Field

## Field Configuration
1. Custom Label
2. Placeholder Support
3. Required / Optional Fields
4. Dynamic Options for Dropdown, Radio & Checkbox Fields
5. Live Field Preview

## Form Rendering
1. Dynamic Frontend Form Generation
